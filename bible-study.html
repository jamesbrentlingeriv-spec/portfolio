Quick Navigation

Bible Study Suite is a browser-based app with Bible reading, note-taking, verse context, per-note audio recordings, and per-note songs/lyrics. All data stored locally (IndexedDB/localStorage). Fetches Bible text from bible-api.com.

Features

📖

Bible Reading

Select book/chapter/translation. Prev/next navigation. Verse search (John 3:16). Click verse to insert ref into notes. Red-letter text in Gospels.

📝

Notes

Create, edit, autosave. Sidebar list for quick access. Delete from list. Verse refs in notes auto-detected and shown in sidebar.

🎙️

Per-Note Audio

Microphone recording tied to current note. Playback, download, delete. Stored in IndexedDB.

🎵

Per-Note Songs

Add song title + lyrics. List view/delete/insert into note. Stored with note.

Vanilla JS IndexedDB + localStorage bible-api.com GitHub Pages

Run Locally & Deploy

Live Server: Right-click index.html → Open with Live Server (VS Code ext) → http://127.0.0.1:5500/

GitHub Pages: Push to main → Settings/Pages → Deploy from main/root. Live after 5min.
