Overview

A Luxury Experience, Online

Pal Optical Boutique is a modern, high-end web application designed to reflect the premium nature of the designer brands it carries. The "Gold & Dark" aesthetic runs throughout every component — from the color palette to the motion design — communicating quality before a single word is read.

Built with React 19 + TypeScript on Vite 6, the site is fast, type-safe, and structured for long-term maintainability. Tailwind CSS 4 handles styling with precision, while Motion (formerly Framer Motion) powers seamless page transitions and interactive elements that feel native and polished.
