# Word Witness

Word Witness helps kids learn to read through Bible stories. An educational web
app with interactive Bible narratives that teach phonics, sight words, and
comprehension.

## Features

- Interactive Bible stories for young learners
- Phonics and sight word practice
- Smooth performance across devices
- Live demo: https://word-witness-840075680860.us-west1.run.app/

Built with modern web technologies.
