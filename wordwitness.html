Word Witness

An educational web app that helps kids learn to read through interactive Bible stories — teaching phonics, sight words, and comprehension with smooth performance across all devices.

Try Live App →

🌟 Overview

Word Witness was built with a simple mission: make learning to read joyful for kids by grounding it in stories they care about. Instead of dry phonics drills, children engage with interactive Bible narratives that teach reading skills naturally through context and repetition.

The app runs on Google Cloud Run for smooth, reliable performance across phones, tablets, and desktops — no install required, just open and read.

✨ Features

📚

Interactive Bible Stories

Engaging narratives designed for young learners, with interactive elements that keep kids focused and curious.

🔤

Phonics Practice

Structured phonics exercises woven into story content so kids learn letter sounds in meaningful context.

👁️

Sight Word Training

High-frequency sight words are highlighted and reinforced throughout each story for faster recognition.

🧠

Comprehension Tools

Simple questions and prompts after each story section to check understanding and reinforce retention.

📱

Cross-Device Performance

Smooth, responsive experience on phones, tablets, and desktops — optimized for the devices kids actually use.

Cloud Run Hosted

Deployed on Google Cloud Run for fast, scalable, always-on availability with no cold-start delays.
